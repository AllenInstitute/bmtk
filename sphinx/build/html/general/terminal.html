Using the terminal¶

Navigating files and directories¶

Command

Action

Example

pwd

print working directory

$ pwd

ls

list files and folders

$ ls

cp

copy file

$ cp file.txt copied_file.txt

mv

move file

$ mv file.txt moved/file.txt

rm

remove file

$ rm file.txt

mkdir

make directory

$ mkdir folder

rmdir

remove directory

$ rmdir folder

cd

change directory

$ cd folder

/../

parent directory

$ cd ..

./

current directory

$ cd ./

~/

home directory

$ cd ~/
