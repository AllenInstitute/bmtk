toolbox.plotter_helper.SubPlotter¶

class SubPlotter(ax, exp, pattern, amplitude, mice, depth=None, sigma=10, centroid=True, electrodes=True, ticks=True)[source]¶

Bases: object

The SubPlotter class can be used together with the Plotter class to modify individual subplots. In most cases, it is probably easier to use Plotter’s plot_all(), which automatically calls this class.

Set attributes and call main functions

Parameters:

  • ax – (maplotlib Axes object) Axes of the subplot.

  • exp – (int/str) Experiment name.

  • patterns – (int/str or list thereof) Pattern names.

  • amplitudes – (int/str or list thereof) Amplitudes in uA.

  • mice – (int/str or list thereof) Mouse names.

  • depth – (tuple or list with 2 elements) If not None, only include spikes of neurons that are in the layer between depth[0] and depth[1]. Defaults to None

  • sigma – (int) Standard deviation of the Gaussian smoother. Defaults to 10.

  • centroid – (bool) If True, plot centroid. Defaults to True.

  • electrodes – (bool) If True, plot electrodes. Defaults to True.

  • ticks – (bool) If True, set ticks to default value. Defaults to True

get_image()[source]¶

Create an intensity map of neural activity based on the number of spikes. Spikes are binned into 1 um^2 pixels and smoothed with a Gaussian filter. The area of the square plot outside of the neural column is set to NaN.

plot_electrodes(**kwargs)[source]¶

Plot the electrode locations as specified in pattern.csv. Electrodes with positive and negative amplitudes are plotted in a different colour.

Parameters:

**kwargs – Passed to ax.scatter().

plot_image()[source]¶

Plot the intensity map of neural activity created by get_image(). The intensities are converted to an image with a green color map. Inactive background is set to black and the NaN values are set to white.

Set axes ticks. Defaults to the 3x3 electrode grid locations

Parameters:

show_ratio()[source]¶

Show the ratio between the centroid-to-electrode distances. Only makes sense in cases where two electrodes stimulate with opposite polarity.
