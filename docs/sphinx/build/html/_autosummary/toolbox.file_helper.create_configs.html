toolbox.file_helper.create_configs¶

create_configs(template, exp, patterns, amplitudes, mice, overwrite=False)[source]¶

Create config files according to specified stimulation parameters. All input parameters will automatically be converted to <parameter>_<name>.

Parameters:

  • template – Template config.json that is modified to generate all other files. For each experiment, it is generally a good idea to manually configure one config.json file. If that works, this function will then do the rest.

  • exp – Experiment name. int/str.

  • patterns – Pattern names. int/str or list thereof.

  • amplitudes – Amplitudes in uA. int/str or list thereof.

  • mice – Mouse names. int/str or list thereof.

  • overwrite – Overwrite existing files if True. defaults to False.

Returns:

None

Example:
Passing arguments 1, ‘1’, [1], [‘1’], ‘pattern1’, [‘pattern1’], ‘pattern_1’, [‘pattern_1’] will all be formatted to [‘pattern_1’]
