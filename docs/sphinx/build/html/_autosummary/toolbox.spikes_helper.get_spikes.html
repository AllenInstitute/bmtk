toolbox.spikes_helper.get_spikes¶

get_spikes(nodes_dirs, spikes_dirs, spikes_bkg_dirs, radius=None, depth=None, v1=True, **kwargs)[source]¶

Get spikes and node positions from network and output files.

Parameters:

  • nodes_dirs (path or list thereof) – directories that point to network/nodes.h5

  • spikes_dirs (path or list thereof) – directories that point to output/spikes.csv

  • spikes_bkg_dirs (path or list thereof) – directories that point to output/bkg/spikes.csv

  • radius (int, optional) – defaults to None.

  • depth (int, optional) – defaults to None.

  • v1 (bool, optional) – defaults to True.

Returns:

node positions

Return type:

ndarray
